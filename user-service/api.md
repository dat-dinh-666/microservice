# APIs:

### User:
  - `POST /users`: Create an user
  - `GET /users/:id`: Get an user
  - `PUT /users/:id`: Update an user (full)
  - `PATCH /users/:id`: Update an user (partial)

### Authentication:
  - `POST /login`: Login using credentials and returns user info
  - `POST /logout`: Logout
  - `GET /forgot-password`: Forgot password
